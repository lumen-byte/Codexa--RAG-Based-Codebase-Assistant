## RAG-Based - Codebase-Assistant ##
A Retrieval-Augmented Generation (RAG) application that helps developers understand and navigate large codebases. The system indexes source code, retrieves relevant files and code snippets based on user queries, and uses AI to generate accurate, context-aware answers.

Built with FastAPI and modern Python tooling, it enables efficient code exploration, architecture understanding, and developer onboarding. Users can ask questions about project structure, functionality, dependencies, and implementation details without manually searching through the entire repository.

This project demonstrates the practical use of RAG techniques for software engineering and code intelligence workflows.
